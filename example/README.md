# Collapsible Tabs Expo Example

Expo SDK 56 example app for `react-native-collapsible-tabs-reanimated`.

The app demonstrates `CollapsibleTabs.ScrollView` inside two collapsible pager tabs.

## Get started

1. Install dependencies

```bash
yarn install
```

2. Start the app

```bash
yarn start
```

In the output, you'll find options to open the app in a development build, emulator, simulator, Expo Go, or web browser.

Edit `src/app/index.tsx` to adjust the demo content.
